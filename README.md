## 기능 소개

- 방 생성
- 실시간 채팅
- 실시간 테이블 연동
- 게임
    - 물체 생성: 빈 공간에 오른쪽 클릭
    - 물체 들기: 왼쪽 클릭
    - 물체 상세: 물체에 오른쪽 클릭
    - 묶음에서 하나 꺼내기: 더블 클릭
    - 뒤집기: 더블 클릭
